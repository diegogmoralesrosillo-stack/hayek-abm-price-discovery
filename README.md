## Overview

This repository contains the simulation code for the paper *"Information Without Coordination: An Agent-Based Model of Hayekian Price Discovery"*.

The model investigates the relationship between informational transmission, coordination, and epistemic diversity in market systems. Three institutional regimes are compared using a minimal agent-based model (ABM):

* **Experiment A** — Market baseline (price adjustment + epistemic selection active)
* **Experiment B** — Price freeze (administered price, dynamic information channel disabled)
* **Experiment C** — No bankruptcy (price adjustment active, Kirznerian selection disabled)

The central finding is the structural decoupling of information and coordination: informational transmission through prices does not imply greater price-fundamental coordination. A price freeze achieves high coordination by constraint while transmitting zero dynamic information.

## Replication

All simulations use deterministic pseudo-random number generation with a fixed base seed (`seed\_base=12345`). Running the notebook from top to bottom reproduces all figures and the summary statistics table reported in the paper.

The key informational metric — Normalized Mutual Information (NMI) — is computed on **first differences** to avoid spurious correlation from shared trends:

```
NMI = I(Δtheta\_t ; Δprice\_t) / H(Δprice\_t)
```

Under price-freeze (Experiment B), Δprice\_t = 0 implies H(Δprice\_t) = 0 and therefore NMI\_B = 0 as a mathematical identity. This is handled explicitly in the `nmi\_discrete()` function.
